# Logo 文件说明

请将以下文件放置在此目录：

## 必需文件
- `logo.png` - 主要logo文件（推荐尺寸：120x40px 或 200x60px）
- `favicon.ico` - 浏览器图标（16x16px 或 32x32px）

## 可选文件  
- `logo-dark.png` - 暗黑模式logo
- `logo-small.png` - 小尺寸logo

## 支持格式
- PNG（推荐，支持透明背景）
- JPG/JPEG
- SVG
- ICO（仅favicon）

## 使用说明
1. 将logo文件复制到此目录
2. 确保文件名与配置中的路径一致
3. 重启Django服务器生效
